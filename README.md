## 어려웠던 점

- 공통된 요소 제거
  - 중복되는 요소를 동적으로 생성하여 공용화하였고, 하나씩 맵핑시켜줘야했던 이벤트 리스너를 함수 내에서 독립적으로 실행할 수 있었습니다.
- 전역 변수 제거
  - 전역 변수처럼 쓰고 있지만 좀더 스코프를 좁혀 각각의 ColumnSection에서만 동작하도록 하였습니다. 이를 통해 칼럼마다 추가 여부 상태를 가질 수 있게 되었습니다.
